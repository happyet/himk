# himk
2 colums simple typecho theme

## 双栏简洁 typecho 主题 HIMK
- 自适应
- 自定义导航
- 后台设置 logo
- 简洁社交信息 微信、qq、微博
- 后台设置右侧头像和一句话
- 支持more截断和自动摘要
- 评论头像支持数字qq邮箱地址自动获取qq头像，其他为cravatar头像
- 无插件支持评论表情
- 自定义侧边栏小工具
- 网友评论血条（活跃评论）
- 显示头像最新评论
- 单页宽屏模板
- 前台登录、sitemap，在页脚
- 文章宽屏图片
- 带链接文章图片 thickbox 放大功能
- 。。。。。
